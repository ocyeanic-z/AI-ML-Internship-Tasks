# Task 3: Heart Disease Prediction

## Objective
To build a machine learning model that predicts whether a person has heart disease based on medical attributes.

## Steps Performed
- Loaded and explored dataset
- Handled missing values (if any)
- Performed Exploratory Data Analysis (EDA)
- Trained classification model (Logistic Regression / Decision Tree)
- Evaluated model using accuracy, confusion matrix, and ROC curve

## Libraries Used
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

## 📈 Outcome
Developed a classification model to predict heart disease risk with evaluation metrics.
